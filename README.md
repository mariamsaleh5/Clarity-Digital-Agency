# Clarity Digital Agency Landing Page UI

A responsive landing page UI for a digital agency, designed to showcase services, technology stack, client testimonials, and a contact form.

## Features
- Hero section with company tagline and stats
- Services overview with Web, Mobile, Cloud, AI, Marketing, and UI/UX offerings
- Technology stack showcase (Frontend, Backend, Mobile, Cloud, Database)
- Client testimonials section
- FAQ section
- Contact form for project inquiries
- Fully responsive layout for desktop, tablet, and mobile
- Built with **Bootstrap 5** for rapid development and consistent design

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Responsive design principles

##  Design Concept
This project simulates a **professional digital agency website**, focusing on:
- Clean, modern layouts
- Structured content presentation
- Mobile-first responsiveness
- Engaging visual hierarchy
